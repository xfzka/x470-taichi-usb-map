## 备注(Note)：

Catalina 系统下，鼠标键盘（2.0）插在**XHC0**下无法休眠。

In Catalina, when USB 2.0 mouse and keyboard plug in XHC0, can't sleep.

建议直接插在PTXH下。

Recommended plug in to PTXH directly。
